# Sean's Place

A retro BBS-themed personal website built with Astro.

## About

Personal site featuring a 90s bulletin board system aesthetic with:
- Classic DOS color palette
- ASCII art and retro design elements
- Blog for writing and sharing thoughts
- Project showcase

## Tech Stack

- **Astro 4.0** - Static site framework
- **TypeScript** - Type safety
- **Content Collections** - Markdown-based blog system

## Development

```bash
npm install
npm run dev
```

Visit `http://localhost:4321`

## Build

```bash
npm run build
npm run preview
```
